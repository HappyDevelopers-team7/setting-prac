## 📌 불필요한 alert 는 토스트 팝업으로 대체

![image](https://github.com/HappyDevelopers-team7/team7-taskify/assets/134386378/a47eddd5-2af2-419f-a41f-813cdda83114)


- 불필요한 alert로 사용자 동작이 추가될 필요 없도록 토스트 팝업으로 대체하였습니다.
